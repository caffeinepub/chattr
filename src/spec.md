# Specification

## Summary
**Goal:** Limit the display width of inline images and GIFs in chat messages to 300px.

**Planned changes:**
- Set max-width to 300px for inline images and GIFs in their non-expanded state in the MessageBubble component

**User-visible outcome:** Inline images and GIFs in chat messages will display at a maximum width of 300px when not expanded, making the chat interface more compact and consistent.
